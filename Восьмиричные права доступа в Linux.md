*   **–í—ã–≤–æ–¥:** –í –∫–æ–Ω—Ç–µ–∫—Å—Ç–µ **–≤–æ—Å—å–º–µ—Ä–∏—á–Ω—ã—Ö –ø—Ä–∞–≤ –¥–æ—Å—Ç—É–ø–∞** (`chmod 755`), –ø–æ–¥ "–¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–º–∏ –±–∏—Ç–∞–º–∏" –ø–æ–Ω–∏–º–∞—é—Ç—Å—è **–¢–û–õ–¨–ö–û `setuid`, `setgid` –∏ `sticky bit`**. –†–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã –∏ ACL ‚Äî —ç—Ç–æ –º–µ—Ö–∞–Ω–∏–∑–º—ã –¥—Ä—É–≥–æ–≥–æ —É—Ä–æ–≤–Ω—è, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ —É–ø—Ä–∞–≤–ª—è—é—Ç—Å—è —á–µ—Ä–µ–∑ –≤–æ—Å—å–º–µ—Ä–∏—á–Ω—É—é –∑–∞–ø–∏—Å—å `chmod`.

### **37. –ö–∞–∫ setuid –≤–ª–∏—è–µ—Ç –Ω–∞ –∏—Å–ø–æ–ª–Ω—è–µ–º—ã–µ —Ñ–∞–π–ª—ã?**

*   **–ü—Ä–∞–∫—Ç–∏–∫–∞:**

    ```bash
    # –ü–†–ò–ú–ï–† 1: –ë–∏–Ω–∞—Ä–Ω—ã–π —Ñ–∞–π–ª (setuid –†–ê–ë–û–¢–ê–ï–¢)

    # –ü–†–ò–ú–ï–† 2: Shell-—Å–∫—Ä–∏–ø—Ç (setuid –ù–ï –†–ê–ë–û–¢–ê–ï–¢ - –ù–ï –î–ï–õ–ê–ô–¢–ï –¢–ê–ö!)

    # –ü–†–ê–í–ò–õ–¨–ù–´–ô –°–ü–û–°–û–ë –¥–ª—è —Å–∫—Ä–∏–ø—Ç–æ–≤:
    # –ù–∞—Å—Ç—Ä–æ–π—Ç–µ sudo —á–µ—Ä–µ–∑ visudo –¥–ª—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ —Å–∫—Ä–∏–ø—Ç–∞
    ```
*   **–í—ã–≤–æ–¥:** `setuid` –¥–∞–µ—Ç —Å–∏–ª—É –≤–ª–∞–¥–µ–ª—å—Ü–∞ —Ñ–∞–π–ª–∞, –Ω–æ **—Ç–æ–ª—å–∫–æ –¥–ª—è —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –ø—Ä–æ–≥—Ä–∞–º–º**. –î–ª—è —Å–∫—Ä–∏–ø—Ç–æ–≤ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `sudo`.
